# 反转字符串

## 题目

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 `s` 的形式给出。

不要给另外的数组分配额外的空间，你必须**[原地](https://baike.baidu.com/item/原地算法)修改输入数组**、使用 `O(1)` 的额外空间解决这一问题。

**示例 1**：

```sh
输入：s = ["h","e","l","l","o"]
输出：["o","l","l","e","h"]
```

**示例 2**：

```
输入：s = ["H","a","n","n","a","h"]
输出：["h","a","n","n","a","H"]
```

**提示**：

- `1 <= s.length <= 10⁵`
- `s[i]` 都是 [ASCII](https://baike.baidu.com/item/ASCII) 码表中的可打印字符

## 思路

在反转链表中，使用了双指针法。

而反转字符串依然是使用反转字符串的方法，但是对于字符串的反转相较于链表更加简单。

因为字符串也是一种数组，因此元素在内存中是连续分布的。

对于字符串，定义两个指针 `left`、`right`，一个在字符串头部，一个在字符串末尾，两个指针同时向中间移动并交换元素。

Go 代码如下：

```go
func reverseString(s []byte) {
	left := 0
	right := len(s) - 1
	for left < right {
		s[left], s[right] = s[right], s[left]
		left++
		right--
	}
}
```

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
